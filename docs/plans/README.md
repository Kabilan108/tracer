## Conventions

- PageBin viewer URLs are capability links (anyone holding one can read the
  live artifact), so only artifact IDs are committed here. Recover a viewer
  URL locally with `pagebin show <id>` or `pagebin receipts`.
- HTML artifacts are committed alongside their PageBin publication; plain
  markdown documents are simply committed.
- Update the published artifact with `pagebin update <file>` after meaningful
  changes so the committed file and the artifact stay in sync.
